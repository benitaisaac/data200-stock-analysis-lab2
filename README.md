# Stock Portfolio Manager 

## Overview 
### This project is a stock portfolio management system with both a console interface and a Tkinter GUI. Users can manage stocks, track daily price/volume data, scrape data from Yahoo Finance, generate reports, display charts, and save/load using a SQLite database.

## Features
* Add, list, and update stocks (buy/sell)

* Add daily stock data manually or via CSV import

* Retrieve stock data from Yahoo Finance (web scraping)

* Generate portfolio reports

* Display stock price charts (Matplotlib)

* Save/load portfolio using SQLite

* Full GUI version included 


## How to Run 

### Console version: 
```
python stock_console.py
```

### GUI Version 
```
python stock_GUI.py

```
